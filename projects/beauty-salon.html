Overview

A clean, modern landing page for a beauty salon. Focused on strong visual hierarchy, responsive layout, and a simple conversion path (services → gallery → contact).

Key Features

  • Mobile-first responsive layout
  • Clear services section and gallery layout
  • CTA buttons that guide the user to contact
  • Clean spacing, typography and consistent components

What I focused on

  • Readable layout and section rhythm
  • Reusable UI patterns (cards, buttons, spacing)
  • Real-world business presentation
